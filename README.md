# Sentiment Analysis NLP Project

This project implements a complete sentiment analysis pipeline using multiple machine learning and deep learning models.

## 📌 Overview
The goal of this project is to classify text into three sentiment classes:
- Negative
- Neutral
- Positive

## ⚙️ Methods Used
- Text preprocessing (cleaning, normalization)
- Bag-of-Words (Unigram features)
- Logistic Regression (baseline model)
- Recurrent Neural Network (RNN)
- Bidirectional LSTM (BiLSTM)
- BERT (Transformer-based model)

## 📊 Dataset
- TweetEval Sentiment Dataset (3-class sentiment classification)

## 🧠 Key Concepts
- Feature engineering
- Class imbalance handling
- Sequence modeling
- Transformer-based NLP

## 📁 File
- `Sentiment_Analysis_Comparisons.ipynb` → Full implementation

## 🚀 Summary
This project demonstrates the progression from traditional machine learning models to advanced deep learning and transformer-based approaches for sentiment analysis.
